# JSON
## java script object notation
### Data representation format very similar to xml eml
### lightweight easy to use read/write easy
### commonly use in api and config files
### json is subset of java script
### integrate easily with other programming language
### datatypes string numbers(int , decimals , sci notation)  booleans null arrays objects
### key value should surround by two quotes quotation marks and  and there pair end with an , comma

# eg:- 

``` json
{
	"key" : "value",
	"key" : "value"


} ```
